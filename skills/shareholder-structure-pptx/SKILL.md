# 株主構成・役員構成 PowerPoint ジェネレーター

BDD（ビジネスデュー・ディリジェンス）において、対象会社の株主構成と役員構成を
上下2セクションのテーブル形式で1枚のPowerPointスライドに整理するスキル。

---

## スライド構成

| セクション | 位置 | 内容 |
|---|---|---|
| **メインメッセージ** | 最上部 | 最大70文字。「〜すべき」で締める |
| **チャートタイトル** | メインメッセージ直下 | 「対象会社概要：株主構成」等 |
| **株主構成テーブル** | コンテンツエリア上半分 | #, 株主名, 役職, 関係, 持株数, 議決権比率(%), 備考 |
| **役員構成テーブル** | コンテンツエリア下半分 | #, 氏名, 役職, 関係, 役員報酬, 備考 |
| **出典** | 左下 | 情報ソースの記載 |

### 株主構成テーブル

| 列 | 内容 | 備考 |
|---|---|---|
| # | 株主番号（1〜N） | |
| 株主 | 株主名（氏名） | |
| 役職 | 会社での役職（無し、代表取締役、取締役等） | |
| 関係 | 創業者との関係（本人、子、配偶者、孫、義子等） | |
| 持株数 | 保有株数（「XXX株」形式） | 右寄せ |
| 議決権比率(%) | 議決権比率（小数点1桁） | 右寄せ |
| 備考 | 補足情報（本件依頼者・創業者 等） | |

- 最終行は「合計」行として持株数合計と議決権比率100.0を表示
- ヘッダー行は黄色系背景（#FFFFCC）

### 役員構成テーブル

| 列 | 内容 | 備考 |
|---|---|---|
| # | 役員番号（1〜N） | |
| 氏名 | 役員名 | |
| 役職 | 代表取締役、取締役等 | |
| 関係 | 創業者との関係 | |
| 役員報酬 | 報酬額（「XX,XXX千円」形式） | 右寄せ |
| 備考 | 補足情報（譲渡後継続、報酬要相談 等） | |

- ヘッダー行は黄色系背景（#FFFFCC）

## 入力パターンと処理フロー

### パターンA：IMが入力された場合

**いきなりPowerPointを作成しない**

1. **Step 1: IMから株主構成・役員構成情報を抽出**し、テーブル項目をドラフトする
2. **Step 2: Markdownでユーザーに提示**し、確認・修正を求める
3. **Step 3: ユーザーの承認後**、PowerPointを生成する

### パターンB：データが整理済みで入力された場合

1. **Step 1: 内容を確認**し、株主・役員データが揃っているか確認
2. **Step 2: 確認後、PowerPointを生成する**

---

## Step 1: IMから株主構成・役員構成情報を抽出する

### IMがPDFの場合の読み取り手順

1. pdf-reading スキルを活用してIMのPDFを読み取る
2. 「株主構成」「株主一覧」セクションを探し、株主名・役職・関係・持株数・議決権比率・備考を抽出
3. 「役員構成」「役員一覧」セクションを探し、氏名・役職・関係・役員報酬・備考を抽出

### 抽出のポイント

- **持株数**: 「XXX株」の数値部分を正確に抽出
- **議決権比率**: 小数点1桁の%値を正確に抽出
- **役員報酬**: 千円単位の数値を正確に抽出（0千円の場合も記載）
- **関係**: 「本人」「子」「配偶者」「孫」「義子」「甥」「社外」等を正確に抽出
- **備考**: 「本件依頼者・創業者」「譲渡後継続、報酬要相談」等の補足情報を正確に抽出
- **合計行**: 株主構成テーブルの最終行に合計持株数と議決権比率100.0を含める

**Chart Title（10〜20文字）**
対象会社概要：株主構成

### ■株主構成
| # | 株主 | 役職 | 関係 | 持株数 | 議決権比率(%) | 備考 |
|---|------|------|------|--------|---------------|------|
| 1 | 藤原 康弘 | 無し | 本人 | 548株 | 54.8 | 本件依頼者・創業者 |
| 2 | 藤原 健一 | 代表取締役 | 子 | 156株 | 15.6 | |
| ... | ... | ... | ... | ... | ... | ... |
| | 合計 | | | 1,000株 | 100.0 | |

### ■役員構成
| # | 氏名 | 役職 | 関係 | 役員報酬 | 備考 |
|---|------|------|------|----------|------|
| 1 | 藤原 健一 | 代表取締役 | 子 | 22,720千円 | 譲渡後継続、報酬要相談 |

インフォメーション・メモランダム

確認メッセージ例：
> 上記の株主構成・役員構成整理でよろしいでしょうか？修正があればお知らせください。確認後にPowerPointを生成します。

## Step 3: PowerPointの生成

### JSONデータ仕様

`{{WORK_DIR}}/shareholder_structure_data.json` に以下の形式で保存する：

```json
{
  "main_message": "対象会社は創業者一族が100%株式を保有するオーナー企業であり、ガバナンス構造を評価すべき",
  "chart_title": "対象会社概要：株主構成",
  "source": "インフォメーション・メモランダム",
  "shareholders": {
    "section_title": "株主構成",
    "columns": ["#", "株主", "役職", "関係", "持株数", "議決権比率(%)", "備考"],
    "rows": [
      {"number": 1, "name": "藤原 康弘", "position": "無し", "relation": "本人", "shares": "548株", "voting_ratio": 54.8, "note": "本件依頼者・創業者"},
      {"number": 2, "name": "藤原 健一", "position": "代表取締役", "relation": "子", "shares": "156株", "voting_ratio": 15.6, "note": ""},
      {"number": 3, "name": "藤原 美紀", "position": "社外", "relation": "配偶者", "shares": "110株", "voting_ratio": 11.0, "note": ""}
    ],
    "total": {
      "shares": "1,000株",
      "voting_ratio": 100.0
    }
  },
  "directors": {
    "section_title": "役員構成",
    "columns": ["#", "氏名", "役職", "関係", "役員報酬", "備考"],
    "rows": [
      {"number": 1, "name": "藤原 健一", "position": "代表取締役", "relation": "子", "compensation": "22,720千円", "note": "譲渡後継続、報酬要相談"},
      {"number": 2, "name": "荒川 真沙美", "position": "取締役", "relation": "子", "compensation": "0千円", "note": "経理担当、使用人分給与のみ、経営管理責任者資格あり"}
    ]
  }
}
```

### JSONフィールド仕様

| フィールド | 型 | 必須 | 説明 |
|---|---|---|---|
| `main_message` | string | 必須 | メインメッセージ。最大70文字、「〜すべき」で締める |
| `chart_title` | string | 任意 | チャートタイトル。デフォルト「対象会社概要：株主構成」 |
| `source` | string | 任意 | 出典テキスト（左下に表示） |
| `shareholders.section_title` | string | 任意 | 株主構成セクションタイトル。デフォルト「株主構成」 |
| `shareholders.columns` | array | 任意 | 列ヘッダー。デフォルト ["#", "株主", "役職", "関係", "持株数", "議決権比率(%)", "備考"] |
| `shareholders.rows` | array | 必須 | 株主データの配列 |
| `shareholders.rows[].number` | int | 必須 | 株主番号 |
| `shareholders.rows[].name` | string | 必須 | 株主名 |
| `shareholders.rows[].position` | string | 必須 | 役職 |
| `shareholders.rows[].relation` | string | 必須 | 関係 |
| `shareholders.rows[].shares` | string | 必須 | 持株数（「XXX株」形式） |
| `shareholders.rows[].voting_ratio` | number | 必須 | 議決権比率(%) |
| `shareholders.rows[].note` | string | 任意 | 備考 |
| `shareholders.total` | object | 任意 | 合計行データ |
| `shareholders.total.shares` | string | 必須 | 合計持株数 |
| `shareholders.total.voting_ratio` | number | 必須 | 合計議決権比率（通常100.0） |
| `directors.section_title` | string | 任意 | 役員構成セクションタイトル。デフォルト「役員構成」 |
| `directors.columns` | array | 任意 | 列ヘッダー。デフォルト ["#", "氏名", "役職", "関係", "役員報酬", "備考"] |
| `directors.rows` | array | 必須 | 役員データの配列 |
| `directors.rows[].number` | int | 必須 | 役員番号 |
| `directors.rows[].name` | string | 必須 | 氏名 |
| `directors.rows[].position` | string | 必須 | 役職 |
| `directors.rows[].relation` | string | 必須 | 関係 |
| `directors.rows[].compensation` | string | 必須 | 役員報酬（「XX,XXX千円」形式） |
| `directors.rows[].note` | string | 任意 | 備考 |

### スクリプト実行コマンド

```bash
pip install python-pptx -q --break-system-packages

python <SKILL_DIR>/scripts/fill_shareholder_structure.py \
  --data {{WORK_DIR}}/shareholder_structure_data.json \
  --template <SKILL_DIR>/assets/shareholder-structure-template.pptx \
  --output {{OUTPUT_DIR}}/ShareholderStructure_output.pptx
```

※ `<SKILL_DIR>` は実際のスキルインストールパスに置き換えること。

### 出力確認

```bash
python -m markitdown {{OUTPUT_DIR}}/ShareholderStructure_output.pptx
```

## デザイン仕様

### フォントサイズ一覧

| 要素 | サイズ | 備考 |
|---|---|---|
| メインメッセージ | 24pt Bold | テーマフォント継承（Meiryo UI / Arial） |
| チャートタイトル | テーマ継承 | テーマフォント継承 |
| セクションタイトル（■株主構成 / ■役員構成） | 14pt Bold | 左寄せ |
| テーブルヘッダー | 12pt Bold | 黄色背景（#FFFFCC） |
| テーブル本文 | 11pt | 左寄せ（名前・役職・関係・備考）、右寄せ（持株数・議決権比率・役員報酬） |
| 合計行 | 12pt Bold | 上線太め |
| 出典 | 10pt | グレー(#666666) |

### 色

| 要素 | カラーコード |
|---|---|
| テキスト（通常） | #333333 |
| テーブルヘッダー背景 | #F5F0D0（テンプレート準拠ベージュ） |
| テーブルヘッダー下線 | #CCCCCC |
| 奇数行背景 | #FFFFFF |
| 偶数行背景 | #FAFAFA |
| 合計行背景 | #F5F5F5 |
| セクションタイトル | #333333 |
| 出典テキスト | #666666 |

### フォントサイズの動的調整

株主数・役員数に応じてフォントサイズを動的に調整する：

| 合計行数 | テーブル本文 | ヘッダー | セクションタイトル |
|---|---|---|---|
| 〜12行 | 13pt | 13pt Bold | 15pt Bold |
| 13〜16行 | 12pt | 12pt Bold | 14pt Bold |
| 17〜20行 | 11pt | 11pt Bold | 13pt Bold |
| 21行〜 | 10pt | 10pt Bold | 12pt Bold |
