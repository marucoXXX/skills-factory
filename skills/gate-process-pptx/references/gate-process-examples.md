# ゲートプロセス記述の良い例・悪い例

## テーマ：M&Aターゲット企業の選定（5フィルター）

### ✅ 良い例

**Main Message**: 5段階のスクリーニングにより有望3社を特定し、デューデリジェンスを開始すべき

**Chart Title**: M&Aターゲット選定プロセス

**Filter 1: 市場適合性**
- Feature 1: 対象市場規模が500億円以上かつ年率5%以上の成長市場に属する企業
- Feature 2: 当社の既存事業領域とのシナジーが定量的に見込める業種

**Filter 2: 財務健全性**
- Feature 1: 直近3期連続で営業利益率10%以上を維持している
- Feature 2: 純有利子負債/EBITDA倍率が3倍以下

**Filter 3: 技術・IP資産**
- Feature 1: 当社が未保有の特許を10件以上保有し、技術的補完関係がある
- Feature 2: R&D投資比率が売上高の5%以上

**Filter 4: 組織・文化適合性**
- Feature 1: 従業員規模500名以下でPMI（統合後管理）リスクが限定的
- Feature 2: 経営陣がM&A後のリテンション契約に前向きな意向を示している

**Filter 5: バリュエーション**
- Feature 1: EV/EBITDA倍率が業界平均の12倍以下で取得可能
- Feature 2: 当社の取得予算上限200億円以内に収まる

**意味合い**
1. 初期ロングリスト50社から5段階のスクリーニングで3社に絞り込まれ、各社とも戦略的適合性が高い
2. 財務・バリュエーション基準を通過した3社はいずれも200億円以内で取得可能であり、資金調達の観点からも実行可能性が高い
3. 次のステップとして、3社に対し秘密保持契約を締結の上、フェーズ1デューデリジェンスを4週間以内に開始すべき

## テーマ：新規サプライヤーの選定（3フィルター）

### ✅ 良い例

**Main Message**: 品質・コスト・供給体制の3基準で候補5社を2社に絞り込み、試験発注を開始すべき

**Chart Title**: サプライヤー選定基準

**Filter 1: 品質基準**
- Feature 1: ISO 9001認証取得済みかつ過去3年間の不良率が0.5%以下
- Feature 2: 当社品質検査チームによるオンサイト監査に合格

**Filter 2: コスト競争力**
- Feature 1: 現行サプライヤー比で単価10%以上のコスト削減が見込める
- Feature 2: 為替変動リスクを含む総調達コスト（TCO）が予算内に収まる

**Filter 3: 供給安定性**
- Feature 1: 月産キャパシティが当社需要の150%以上をカバーできる
- Feature 2: BCP体制（複数生産拠点・在庫バッファ）が整備されている

**意味合い**
1. 品質基準で候補5社中3社が残り、コスト基準でさらに2社に絞り込まれた
2. 最終候補2社はいずれも供給安定性を満たしており、単一障害点リスクが低い
3. 来月中に両社へ試験発注（各100ロット）を実施し、実績ベースで最終1社を選定すべき

---

### ❌ 悪い例（よくある間違い）

**フィルター（NG: 曖昧・抽象的）**

Filter 1: 良い会社かどうか ← 基準が不明確
- Feature 1: 評判が良い ← 定量基準がない
- Feature 2: 信頼できる ← 主観的

Filter 2: コスト ← フィルター名が一語で抽象的
- Feature 1: 安い ← 何と比較して安いか不明
- Feature 2: 予算に合う ← 具体的な数値基準がない

**意味合い（NG: 曖昧・当たり前）**
1. いくつかの会社が残った ← 「いくつか」は具体性ゼロ
2. 良い候補が見つかった ← 何が「良い」か不明
3. 引き続き検討する ← アクションではなく状態
