# ロジックツリー記述の良い例・悪い例

## テーマ：新規海外市場への進出判断

### ✅ 良い例

**Main Message**
当社は東南アジア市場への早期進出に向けた体制構築を開始すべき

**Chart Title**
東南アジア市場進出の妥当性

**Main Logic: 東南アジア進出は事業成長の最重要ドライバーである**
1. 市場環境が当社の強みと高い親和性を持つ
2. 競合に先行できるタイミングが限られている
3. 自社リソースの転用可能性が高い

**Sub Logic 1: 市場の魅力度が高い**
1. ASEAN主要5カ国のデジタル広告市場は2024年に前年比25%成長し、140億ドル規模に到達した
2. 現地企業のSaaS導入率はまだ15%にとどまり、日本の45%と比較して大きな成長余地がある
3. インドネシア・ベトナムではスマートフォン普及率が85%を超え、モバイルファーストのサービス需要が急伸している

**Sub Logic 2: 競合環境に参入余地がある**
1. 日系競合A社・B社はいずれもインド・中国に注力しており、東南アジアでの日系プレイヤーは実質ゼロ
2. 現地プレイヤーは資金調達フェーズにあり、エンタープライズ向け機能が未成熟（NPS調査で機能不足が最多不満）
3. 当社が持つ日本語・英語バイリンガル対応は、日系現地法人2,800社への差別化要因になる

**Sub Logic 3: 自社リソースの活用可能性が高い**
1. 既存プロダクトの多言語対応は80%完了しており、ローカライズ追加工数は約3人月と見積もられている
2. シンガポールオフィスに現地市場経験を持つBD人材が4名在籍し、即座にパイロット営業が可能
3. 既存顧客の12%（約45社）が東南アジアに拠点を持ち、初期導入候補としてアプローチ可能

**意味合い**
1. FY2025Q2までにシンガポール拠点を起点としたパイロットプログラムを開始する
2. 既存顧客45社への初期導入アプローチを今四半期中に実施する
3. 現地BD人材4名にパイロット営業の専任体制を組み、KPIを設定する

### ❌ 悪い例（よくある間違い）

**Main Logic: 海外進出は良い（NG: 抽象的すぎる）**
1. 海外は成長している ← 具体性がない、どの地域・市場かが不明
2. 競合が少ない ← どの競合と比較してなのかが不明
3. やればできそう ← ファクトに基づかない感覚論

**Sub Logic 1: 市場が大きい（NG: 曖昧で帰納法が成立しない）**
1. 東南アジアは人口が多い ← 市場規模の根拠としては不十分
2. デジタル化が進んでいる ← 何がどの程度進んでいるのかが不明
3. チャンスがありそう ← 主観的評価であり事実ではない

**帰納法で注意すべきNGパターン:**
- **事実の飛躍**: Sub LogicからMain Logicへのジャンプが大きすぎる（例: 「市場が大きい」→「必ず成功する」）
- **事実の不足**: 2つ以下の事実から無理に結論を導く
- **事実の偏り**: 都合の良い事実だけを集めて結論を導く（確証バイアス）
- **抽象度の不一致**: Sub Logic間で具体性のレベルが揃っていない

## テーマ：社内DX推進の優先度判断

### ✅ 良い例

**Main Message**
全社的なDX推進のため、まず営業部門のCRM刷新を最優先で実行すべき

**Chart Title**
DX推進における優先投資領域

**Main Logic: 営業部門のCRM刷新がDX推進の最大レバーである**
1. 営業部門の非効率が全社業績のボトルネックになっている
2. CRM刷新は短期間で最大のROIが見込める
3. 営業DXの成功が他部門展開のモデルケースになる

**Sub Logic 1: 営業部門の生産性が著しく低い**
1. 営業担当者の40%の時間がデータ入力と報告書作成に費やされており、顧客対面時間は月平均32時間にとどまる
2. 現行CRMの入力項目は128フィールドあり、業界標準（45フィールド）の約3倍で、入力完了率は62%
3. 案件情報のリアルタイム把握ができず、月末に手動集計しているため経営判断が常に1ヶ月遅れている

**Sub Logic 2: CRM刷新のROIが明確である**
1. ベンダー3社の提案を比較した結果、導入コスト1.2億円に対し年間削減効果は2.8億円（回収期間5.1ヶ月）
2. 先行導入した大阪支店では3ヶ月で営業生産性が35%向上し、受注率が12ポイント改善した
3. データ入力の自動化により月80時間の工数が削減され、顧客対面時間を倍増できる見込み

**Sub Logic 3: 他部門への展開効果が期待できる**
1. 営業データが整備されることで、マーケティング部門のリード分析精度が向上し、広告ROASの15%改善が試算される
2. CRM刷新で確立したデータガバナンスの方針は、経理・人事のDXにそのまま適用可能
3. 全社DX推進室が実施した調査で、営業DXの成功事例があれば他部門の協力意欲が2倍になると回答（n=120）

**意味合い**
1. Q1中にCRMベンダー選定を完了し、全社導入のキックオフを実施する
2. 大阪支店の成功事例を標準化し、全支店展開のプレイブックを策定する
3. CRM刷新で得られるデータ基盤を活用し、マーケ・経理のDXロードマップを並行策定する
