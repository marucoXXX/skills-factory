# 財務ベンチマーク比較 PowerPoint ジェネレーター

対象会社と競合N社を**複数の財務指標**で一覧比較する、コンサル品質のベンチマーク・スライド。
Small Multiples（小型チャート並列）方式で、1枚で複数の視点から財務パフォーマンスを俯瞰できる。
BDD（ビジネスデュー・ディリジェンス）・投資判断・経営会議・中期計画策定など、
財務面での競合比較を求められる場面で使用。

---

## スライド構成

| セクション | 位置 | 内容 |
|---|---|---|
| **メインメッセージ** | 最上部 | 最大70文字。ベンチマークからの示唆を「〜すべき」で締める |
| **チャートタイトル** | メインメッセージ直下 | 「財務ベンチマーク：○○社 vs 主要競合4社（FY2024）」 |
| **小型バーチャート × 6** | 中央2×3グリッド | 各指標の水平バーチャート |
| **出典** | 下部 | 情報ソースの記載（年度明記推奨） |

### 小型バーチャート（1指標 = 1チャート）の構成

各セル内部:
- **タイトル**: 指標名 + 単位（例：「売上高（億円）」「ROE（%）」）
- **下線**: タイトル下の横線
- **企業名列**（左）: 企業名（対象会社は赤 Bold）
- **バー**: 水平バー
  - 対象会社: **赤 #E15759**（負の値は濃赤 #8B2C2E）
  - その他: **紺 #4E79A7**（負の値は濃赤 #B83A3A）
- **値ラベル**（右）: 数値（指標に応じたフォーマット）

### 対応指標数

- **デフォルト構成**: 6指標（2行×3列）
- 指標が6未満の場合、左上から順に埋めて余白ができる
- 指標が6超の場合、最初の6つのみ表示（警告あり）

### 推奨される6指標

| カテゴリ | 指標 | 意味 |
|---|---|---|
| 規模 | **売上高** | 事業規模 |
| 成長性 | **売上成長率（3年CAGR）** | 成長スピード |
| 収益性 (PL) | **営業利益率** | 本業の利益性 |
| 収益性 (CF) | **EBITDAマージン** | キャッシュ創出力 |
| 資本効率 | **ROE** | 株主資本収益率 |
| 財務健全性 | **自己資本比率** | 財務の安定性 |

他の代替指標: ROA, Net Debt/EBITDA, D/E ratio, ROIC, 総資産回転率, 粗利率など

## 入力パターンと処理フロー

### パターンA：各社の財務データが部分的に揃っている場合

**いきなりPowerPointを作成しない**

1. **Step 1: 各社の財務データを整理**（欠損値は N/A として許容）
2. **Step 2: Markdownでユーザーに提示**し、確認・修正を求める
3. **Step 3: ユーザーの承認後**、PowerPointを生成する

### パターンB：データが整理済みで入力された場合

1. **Step 1: 内容を確認**し、指標と企業が揃っているか確認
2. **Step 2: 確認後、PowerPointを生成する**

### 財務データの取得ソース

| 企業タイプ | 主な情報源 |
|---|---|
| 上場企業（日本） | EDINET、有価証券報告書、決算短信、各社IR |
| 上場企業（海外） | SEC 10-K, 10-Q, IR、Bloomberg |
| 有償データベース | SPEEDA、日経バリューサーチ、Refinitiv |
| 非上場企業 | 東京商工リサーチ、帝国データバンク（限定的） |

```json
  "target_company": "対象会社",
  "companies": [
    {"name": "対象会社"},
    {"name": "競合A"},
    {"name": "競合B"},
    {"name": "競合C"},
    {"name": "競合D"}
  ],
  "metrics": [
    {
      "name": "売上高",
      "unit": "億円",
      "decimals": 0,
      "higher_is_better": true,
      "sort": "desc",
      "values": {
        "対象会社": 2800,
        "競合A": 4200,
        "競合B": 3500,
        "競合C": 1900,
        "競合D": 1200
      }
    },
    {
      "name": "売上成長率（3年CAGR）",
      "unit": "%",
      "decimals": 1,
      "show_sign": true,
      "sort": "desc",
      "values": {
        "対象会社": 3.2,
        "競合A": 8.5,
        "競合B": 5.1,
        "競合C": 12.4,
        "競合D": -1.3
      }
    }

```

### JSONフィールド仕様

#### トップレベル

| フィールド | 型 | 必須 | 説明 |
|---|---|---|---|
| `source` | string | 任意 | 出典テキスト。年度明記を推奨 |
| `main_message` | string | 必須 | メインメッセージ（最大70文字） |
| `chart_title` | string | 任意 | チャートタイトル |
| `target_company` | string | 任意 | 対象会社名（赤ハイライト対象） |
| `companies` | array | 必須 | 企業定義の配列（4〜6社推奨） |
| `companies[].name` | string | 必須 | 企業名 |
| `metrics` | array | 必須 | 指標の配列（6指標まで、それ以降は無視） |

#### メトリック（metric）

| フィールド | 型 | 必須 | 説明 |
|---|---|---|---|
| `name` | string | 必須 | 指標名（例：「売上高」「ROE」） |
| `unit` | string | 任意 | 単位（例：「億円」「%」）。タイトルに括弧書きで表示 |
| `decimals` | int | 任意 | 小数点以下桁数（デフォルト=1）。売上高などは0を推奨 |
| `show_sign` | bool | 任意 | プラスの数値にも `+` を付けるか（デフォルト=false）。成長率などは true 推奨 |
| `higher_is_better` | bool | 任意 | 高いほど良い指標か（将来の色分け拡張用、現状は未使用） |
| `sort` | string | 任意 | 並び順：`"desc"`(降順、デフォルト推奨) / `"asc"`(昇順) / `"keep"`(companies 配列順) |
| `values` | object | 必須 | `{企業名: 数値}` のマッピング。null/未設定は N/A 表示 |

### 負の値の表示

- 最大絶対値を基準にバー幅をスケーリング
- ゼロラインを**バーエリアの中央**に配置
- 正の値はゼロから**右**に、負の値はゼロから**左**に伸びる
- 値ラベルはバーの外側（正は右、負は左）に配置

## 注意事項

- **企業数**: 4〜6社が最適。少なすぎると比較の説得力低下、多すぎるとバーが細くなる
- **年度の統一**: 全社同じ会計年度のデータを使用する。決算期が異なる場合は明記
- **連結/単体の統一**: 全社連結または全社単体で揃える
- **数値の単位統一**: 「億円」「百万円」など単位は指標ごとに統一（混在NG）
- **成長率の計算基準**: CAGRは同じ期間・手法で計算（例：全社「3年CAGR（FY2021→FY2024）」）
- **N/A値の扱い**: 非上場企業など特定指標が取得不可の場合、null で明示。バーは非表示で「N/A」表示
- **指標数の上限**: 6指標。7個目以降は警告付きで無視される（将来2スライド対応の余地あり）
- **対象会社の検証**: `target_company` と `companies[].name` / `metrics[].values` のキーを完全一致させる

## 競合分析デッキでの使用例

`competitor-analyst-agent` オーケストレータースキルから呼び出される位置づけ:

```
1. 対象会社プロファイル (customer-profile-pptx)
2. 対象会社 事業ポートフォリオ (business-portfolio-pptx)
3. 対象会社 SWOT (swot-pptx)
4. マクロ環境 - PEST分析 (pest-analysis-pptx)
5. 市場環境 - 市場規模推移 (market-environment-pptx)
6. 市場環境 - 市場シェア分析 (market-share-pptx)
7. 市場環境 - ポジショニングマップ (positioning-map-pptx)
8. 競合比較サマリー (competitor-summary-pptx)
9. 財務ベンチマーク ← 本スキル
10. 各競合プロファイル (customer-profile-pptx × N)
```

9枚目（財務ベンチマーク）は、8枚目（定性的サマリー）と10枚目以降（個社詳細）の**橋渡し**として機能。
「各社がどこで強く・どこで弱いか」を定量的に可視化することで、
その後の個社プロファイル（詳細分析）に入る前の全体像を提供する。

### 他スキルとの使い分け

| スキル | 用途 |
|---|---|
| **financial-benchmark-pptx** | **複数財務指標×複数社のマトリクス比較**（1枚で網羅） |
| competitor-summary-pptx | 企業属性（事業・所在地・規模等）の横並び比較 |
| customer-profile-pptx | 1社の詳細プロファイル（概要＋業績チャート） |
| business-portfolio-pptx | 1社のセグメント別売上構成 |
| revenue-analysis-pptx | 1社の売上・EBITDA推移（複数年） |

**financial-benchmark-pptx は「横比較（社×指標）」、他は「縦深掘り（1社の詳細）」** と役割分担する。
