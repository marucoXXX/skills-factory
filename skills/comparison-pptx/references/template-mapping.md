## Shapeマッピング表

| 論理フィールド | Shape名 | 内容 |
|---|---|---|
| main_message | `Title 1` | スライド上部のMain Message（1文、最大70文字） |
| chart_title | `Text Placeholder 2` | Chart Title（10〜20文字） |
| axis_label | `TextBox 6` | 左上の評価軸ラベル（デフォルト「評価軸」） |
| col_a | `TextBox 22` | 1列目パターン名（Aパターン） |
| col_b | `TextBox 3` | 2列目パターン名（Bパターン） |
| col_c | `TextBox 4` | 3列目パターン名（Cパターン） |
| label_1 | `Rectangle 5` | 1行目の評価軸ラベル |
| label_2 | `Rectangle 8` | 2行目の評価軸ラベル |
| label_3 | `Rectangle 9` | 3行目の評価軸ラベル |
| label_4 | `Rectangle 12` | 4行目の評価軸ラベル |
| label_5 | `Rectangle 15` | 5行目の評価軸ラベル |
| implications | `TextBox 29` | 意味合いセクション（Para0=ヘッダー固定, Para1-3=Implication） |
| cell_1_a〜cell_5_c | `TextBox 30〜45` | 各セル（Para0=マーク, Para1=コメント） |

## マーク記号の使い方

| 記号 | 意味 |
|---|---|
| ◎ | 非常に優れている・最も推奨 |
| ○ | 良好・問題なし |
| △ | 課題あり・条件付き |
| × | 不適・非推奨 |
| ー | 該当なし・評価対象外 |
