## 共通 Shape（3矢羽・5矢羽で同じ）

| Shape名 | 種別 | 用途 |
|---|---|---|
| `Title 1` | PLACEHOLDER | Main Message |
| `Text Placeholder 2` | PLACEHOLDER | Chart Title |
| `TextBox 6` | TEXT_BOX | Label1（上段ラベル） |
| `TextBox 7` | TEXT_BOX | Label2（下段ラベル） |
| `Straight Connector 18` | LINE | Label1とLabel2の間の区切り線 |
